The dataset used for this project is titled “Facebook Users by Country Data.” It contains 126 observations (countries) and five variables. Each row represents one country, and each column provides key information about Facebook usage and demographics. The variables are:

Country – The name of the country.

Users – The total number of Facebook users in that country.

FB_Users (%) – The percentage of the country’s population that uses Facebook.

Date – The year or date the data was collected.

Population – The total population of the country.
